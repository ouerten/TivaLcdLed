## This application is used to write text messages on an LCD screen using LEDs and TivaC EK-TM4C123GXL <br />   
![image](https://github.com/ouerten/TivaLcdLed/blob/master/TM4C123GXLpin.jpg?raw=true)   <br /><br /><br />


  Port Connections:
  
  Tiva C    LCD Port  
  PB0  ->   RS  
  PB1  ->   E  
  PD4  ->   D4  
  PD5  ->   D5  
  PD6  ->   D6  
  PD7  ->   D7  

  LED connections  

  Tiva C     LED  
  E0	       Red  
  E1	       White  
  E2         Green  
  E3         Blue
